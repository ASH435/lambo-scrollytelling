# Master Prompt: Lamborghini Immersive Scrollytelling

## 1. High-Level Vision
**Vibe:** A "High-Performance Technical Editorial." 
The goal is to eliminate all generic "empty space" by introducing a dense, cockpit-inspired layout where typography and 3D assets are tightly integrated. The interface should feel like a multi-layered mechanical object, not a flat webpage.

## 2. Structural Requirements
- **Hero Screen:** Full-screen 3D focal point. Massive, tight-tracked "REVUELTO" headline (Geist Sans style) with -0.08em letter-spacing. The title should partially overlap a 3D deconstructed V12 engine.
- **Content Blocks:** Asymmetric 50/50 and 70/30 splits. No centered text. Use "Slotted" layouts where narrative text sits in floating glass panels.
- **Narrative HUD:** Persistent top-bar for telemetry (AERODYNAMICS, HYBRID, PERFORMANCE) and a dedicated right-side "Progress Scrolly-Line."

## 3. Visual Style (The Aerodynamic Monolith)
- **Palette:** Zinc-950 (#09090B) backdrop. Ignition Orange (#FF6600) for all functional highlights, glows, and technical IDs.
- **Materiality (Liquid Glass):** All text containers must use 40% opacity panels with a 20px backdrop-blur and 1px "light-leak" borders on the top/left edges.
- **Density:** 8/10 Cockpit Density. Group specs like "1015 CV", "0-100: 2.5s", and "TOP SPEED: 350+" in high-contrast monospace clusters.

## 4. Specific Screen/Section Redesign:
1. **01_THE_CORE:** Engine deconstruction. Left-aligned technical stats. Floating glass panel on the right.
2. **02_THE_EDGE:** Aero-focus. Grid of 2x2 performance chips showing downforce percentages (66%, 74%).
3. **03_THE_FRAME:** Carbon lattice. End-to-end background image. Right-aligned bold typography.
4. **04_THE_LEGACY:** High-contrast 3-column image grid with grayscale-to-color hover reveals.

## 5. Anti-Patterns to Avoid
- No empty whitespace without technical purpose.
- No centered hero text.
- No generic serif fonts.
- No rounded corners > 4px (stay sharp).
- No emojis or filler UI text ("Scroll to explore").
